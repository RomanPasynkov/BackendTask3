# Task 3 Web

Решение задания выполнено на PHP без фреймворков с использованием PDO и MySQL/MariaDB.

## Что реализовано

- HTML-форма со всеми полями из задания
- CSS-оформление страницы
- серверная валидация всех полей
- сохранение в MySQL/MariaDB
- использование prepared statements
- нормализованная схема базы данных:
  - `submissions` — основная анкета
  - `programming_languages` — справочник языков
  - `submission_languages` — связь многие-ко-многим

## Файлы

- `index.php` — форма, валидация, сохранение
- `styles.css` — оформление
- `schema.sql` — создание БД и таблиц

## Настройка базы данных

1. Создайте базу и таблицы, выполнив `schema.sql`.
2. При необходимости измените настройки подключения в начале файла `index.php`:
   - `DB_HOST`
   - `DB_PORT`
   - `DB_NAME`
   - `DB_USER`
   - `DB_PASSWORD`

## Запуск

Если у вас установлен PHP, можно запустить встроенный сервер:

```bash
php -S localhost:8000
```

После этого откройте в браузере:

```text
http://localhost:8000
```

## Важно

- Для работы нужен включённый драйвер `pdo_mysql`.
- Если база данных не подключена или схема не импортирована, страница покажет сообщение об ошибке сохранения.
- Все основные проверки выполняются на сервере, как требует задание.
